## The MAD Podcast with Matt Turck — Stripe's AI Chief: How AI Agents Will Buy, Sell, and Pay

*The MAD Podcast with Matt Turck*

**Agentic commerce requires entirely new financial infrastructure, not just smarter models, as agents transition from simple shoppers to autonomous economic actors.**

Emily Sense, Head of Data and AI at Stripe, discusses the shift from hypothetical agent commerce to deployed infrastructure. Stripe launched the Agentic Commerce Protocol and Link Wallet to handle agent transactions safely, addressing trust and fraud issues like "token theft," where fraudsters steal API credits rather than money. Sense notes that SaaS pricing models are breaking because AI has real marginal costs, necessitating real-time, usage-based billing. She also highlights "vibe deployment" as the new bottleneck—while AI can write code fast, deploying it still requires navigating fragmented human-centric dashboards. The rise of solopreneurs earning six figures is a direct macroeconomic result of these tools. "It's not Emily permissioning an agent to buy on her behalf. It's Emily has an agent who's tasked with running a business."

Stripe 数据与 AI 负责人 Emily Sense 探讨了 Agent 电商从概念走向实际基础设施的进程。Stripe 推出了 Agentic Commerce Protocol 和 Link Wallet 来安全处理 Agent 交易，解决信任和欺诈问题，比如“Token 盗窃”——欺诈者不再偷钱，而是偷取 API 额度。Sense 指出 SaaS 定价模式正在失效，因为 AI 有真实的边际成本，必须采用实时、基于用量的计费。她还强调了“Vibe Deployment”已成为新的瓶颈——虽然 AI 写代码很快，但部署仍需在碎片化的人类中心化仪表板中周旋。六位数收入的个体创业者激增，正是这些工具带来的宏观经济结果。“不是 Emily 授权 Agent 代她买东西，而是 Emily 有一个 Agent 被指派去经营一家企业。”
