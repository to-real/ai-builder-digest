## Nan Yu

*Head of Product, Linear*

**Nan Yu: Effective AI agents require intentional friction—multi-turn interaction beats one-shot generation to prevent user disengagement.**

Linear found that one-shot project updates created "slop" because users disengaged their brains. Switching to a multi-turn interactive mode where the agent prompts for emphasis and missing context resulted in significantly better updates, proving that introducing friction can yield a net benefit.

Linear 发现一次性生成项目更新会产生“垃圾内容”，因为用户会停止思考。切换到多轮交互模式后，Agent 会提示用户强调重点和补充缺失的上下文，从而产生质量高得多的更新，这证明了引入适当的摩擦可以带来净收益。

## Claude Blog: Claude Code now supports artifacts

**Claude Code Artifacts transforms coding sessions into live, shareable web dashboards, eliminating the need for manual status updates.**

Claude Code now supports Artifacts, which turn work progress into live, shareable visual pages like PR walkthroughs, system explainers, and dashboards. These pages update in real-time as the session works, drawing on full context including codebase and connected tools. Available in beta for Team and Enterprise plans, Artifacts are private to the organization and aim to reduce communication overhead by letting stakeholders view the same live context.

Claude Code 现已支持 Artifacts，它将工作进度转化为实时可共享的可视化页面，例如 PR 演示、系统解释器和仪表盘。这些页面随着会话的进行实时更新，并利用包括代码库和连接工具在内的完整上下文。该功能目前处于 Beta 阶段，面向 Team 和 Enterprise 计划，Artifacts 仅在组织内部可见，旨在让利益相关者查看相同的实时上下文，从而减少沟通开销。
