**Anthropic's Managed Agents architecture decouples the agent logic from the execution environment to solve reliability and scaling issues.**

Anthropic introduced Managed Agents, a system that decouples the "brain" (agent harness) from the "hands" (sandbox execution). This design treats containers as "cattle" rather than "pets," allowing for failure recovery without session loss and reducing time-to-first-token by 60% by provisioning containers only when needed.

Anthropic 推出了 Managed Agents，这是一个将“大脑”（Agent Harness）与“双手”（Sandbox 执行环境）解耦的系统。这种设计将容器视为“牲畜”而非“宠物”，允许在不丢失会话的情况下从故障中恢复，并通过仅在需要时配置容器，将首字延迟降低了 60%。

→ [原文](https://www.anthropic.com/engineering/managed-agents)
